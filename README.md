# IBM - EasyNet
Read the [SRS](docs/easynet.pdf) document for a quick project overview!

## Build
Run `./gradlew build` to build project files. Built JARs are placed in `./build/libs`.  

## Run
Run `./gradlew run` to run EasyNet development server.
